# Machine à Invocation

Petit projet réalisé pendant la formation Ada Tech School.

## Technologies

- HTML5
- CSS3
- JavaScript (DOM)

## Fonctionnalités

- Saisie du prénom
- Choix d'une créature
- Affichage dynamique d'un message
- Masquage du formulaire
- Réinitialisation sans recharger la page

## Concepts JavaScript utilisés

- getElementById()
- addEventListener()
- preventDefault()
- textContent
- classList
- reset()
